# JS Challenger Solution

## Variable

### 1. Reassign a value to a variable

```javascript
let num = 1;
num = 2;
console.log(num);
```

### 2. Assign a value to a variable
- Here, we declare the variable num. But, it has no value yet. Assign a value to it and run the code.
```javascript
let num;
num = 1;
console.log(num);
```
